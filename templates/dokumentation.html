API för Din Röst

Din Rösts API ger dig som utvecklare kort information och åtkomst till Din Rösts data. API:et är utvecklad utifrån RESTs principer. All data hämtas via HTTP/GET och formatet är JSON och XML. Din Röst tillhandahåller inte någon ytterligare support till utvecklare utöver denna information. Inga licenser krävs för användning.

Endpoint

Endpoints är det sista katalognamnet före sista och avslutande ”/v1/riksdagsinfo”. Allt efter detta är detsamma som en ”identifikator”.

HTTP/GET anrop görs till:

/v1/valkretsar

/v1/partier

/v1/ledamoter

/v1/ledamotinfo

Returnerar

  • Diagram som visar Sveriges alla riksdagsledamöter fördelade på respektive partitillhörighet
  • Lista över Sveriges valkretsar. Varje valkrets är ’klickbar’ och länkar till en ny sida

Request parameters

  • Parti
    • returnerar ett diagram över Sveriges riksdagsledamöter uppdelat på partitillhörighet (visas på indexsida)
    • returerar ett diagram över valkretsens riksdagsledamöter uppdelat på partitillhörighet.
  • Valkrets
    • returnerar en lista över valkretsar i Sverige (visas på indexsida). Varje valkrets är ’klickbar’
    • returnerar den valda valkretsen och de riksdagsledamöter som finns i denna valkrets. Varje riksdagsledamot är ’klickbar’.
  • Ledamot
    • returnerar information (namn, partitillhörighet, födelseår, kön, epost och titel)

Exempel på parametrar:

'http://data.riksdagen.se/personlista/?termlista=valkrets&utformat=xml’
'http://data.riksdagen.se/personlista/?fnamn={fnamn}&enamn={enamn}&utformat=json'

Exempel på request

GET /v1/index/data.riksdagen.se/personlista/?utformat=json&valkrets={Blekinge län}

Felmeddelande

Svarskoden visas alltid på webbsidan.

  • 404  Not found!: Hoppsan, sidan kunde inte hittas!
  • 405  Method not allowed!: Metoden är inte tillåten.
